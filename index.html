homelab-nut

Network UPS Tools, set up from your laptop.

Get Started →GitHubRoadmap

Power resilience

Coordinated graceful shutdown across every machine in your homelab when the UPS battery runs low.

Real-time monitoring

Live UPS status across the fleet — battery, load, voltage, runtime — via the native NUT protocol.

Smart notifications

Slack, Discord, Pushover, Telegram, ntfy — alerts when power events happen, so you know before things go dark.

Automated actions

Per-host shutdown recipes for the tricky devices — UniFi, Synology, LG TVs — so you stop scripting them by hand.

What the CLI looks like

In active development — see the roadmap for phase status. Today's bash scripts keep working; the new CLI wraps them in v1, then progressively ports to native Go.

# 1. Describe your hosts interactively
homelab-nut init

# 2. SSH out and configure NUT across the fleet
homelab-nut apply

# 3. Watch the live UPS dashboard
homelab-nut status

# Or just open the full TUI:
homelab-nut
